# Threat Classification Models

Place trained threat classification models here.

## Files
- `random_forest.joblib` - Random Forest classifier
- `gradient_boosting.joblib` - Gradient Boosting classifier
- `deep_model.h5` - Deep neural network
- `label_encoder.joblib` - Label encoder for threat categories
- `scaler.joblib` - Feature scaler
- `model_config.json` - Model configuration
